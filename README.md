# Eisenhower Matrix

Live site: https://mohammedalhaq.github.io/eisenhower-matrix/

A React and TypeScript Eisenhower Matrix app for organizing tasks by urgency and importance.

## Tech Stack

- React
- TypeScript
- Vite
- ESLint
- GitHub Pages

## Requirements

- Node.js 20.19.0 or newer
- npm

## Getting Started

Install dependencies:

```sh
npm install
```

Start the development server:

```sh
npm run dev
```

## Scripts

```sh
npm run dev
```

Starts the local Vite development server.

```sh
npm run build
```

Builds the app, generates the SSR bundle, and prerenders the production output.

```sh
npm run preview
```

Serves the production build locally.

```sh
npm run lint
```

Runs ESLint.

```sh
npm run deploy
```

Deploys the `dist` directory to GitHub Pages.

## Project Structure

```text
src/      React application source
public/   Static assets
scripts/  Build and prerender scripts
dist/     Production build output
```
